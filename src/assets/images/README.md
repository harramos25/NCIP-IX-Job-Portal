# NCIP Logo

Place your NCIP logo image here with the filename: `ncip-logo.png`

**Recommended specifications:**
- Format: PNG (with transparency) or SVG
- Size: Approximately 200x200px or similar aspect ratio
- File name: `ncip-logo.png`

The logo will automatically appear in the header of all pages. If the logo file is not found, it will be hidden gracefully.
